# Regression Metrics

В этом разделе собраны базовые метрики качества для задач регрессии и прогнозирования.
Фокус — на понимании смысла метрик, их ограничений и корректной интерпретации результатов.

## Содержимое

### 1) Базовые метрики регрессии
**Файл:** [basic_metrics.ipynb](basic_metrics.ipynb)

Рассматриваются и реализуются (вручную) основные метрики регрессии:
- **MAE** — средняя абсолютная ошибка
- **MSE** — средняя квадратичная ошибка
- **RMSE** — корень из MSE (ошибка в единицах целевой переменной)
- **R²** — коэффициент детерминации (сравнение с предсказанием среднего)
- **MAPE** — средняя относительная ошибка

> Примечание: `В ноутбуке результаты ручных вычислений сопоставляются с реализациями из `sklearn`
для валидации корректности.`

---

### 2) sMAPE vs MAPE
**Файл:** [smape.ipynb](smape.ipynb)

Показано, почему **MAPE** плохо ведёт себя при малых значениях целевой переменной
(вплоть до взрывного роста), и как **sMAPE** делает оценку более устойчивой.
Добавлена визуализация, демонстрирующая различия поведения метрик около нуля.
