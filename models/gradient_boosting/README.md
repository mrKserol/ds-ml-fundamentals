# Gradient Boosting

В этой задаче построим градиентный бустинг над решающими деревьями и убедимся, что самый популярный алгоритм в классическом машинном обучении под капотом оказывается не таким уж и сложным.

## Что внутри

- `gradientboosting.py` — реализация градиентного бустинга для регрессии.
- `gradient_boosting_from_scatch.ipynb` — пошаговый разбор с обучением, сравнением со `sklearn` и визуализацией качества.
- `users.csv` — учебный датасет.
