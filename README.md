# Data Science & Machine Learning fundamentals

Репозиторий с фундаментальными понятиями Data Science и Machine Learning,
реализованными на Python с акцентом на понимание принципов работы алгоритмов и методов.

Цель репозитория — сформировать прочную теоретическую и практическую базу,
необходимую для прикладной аналитики, машинного обучения и дальнейшей работы
с более сложными моделями. Здесь основной фокус сделан не на использовании
готовых библиотек, а на понимании того, что происходит «под капотом».

## Принципы работы

- Минимальное использование высокоуровневых библиотек
- Явная математическая формулировка перед реализацией
- Разделение данных, логики и оценки моделей
- Акцент на интерпретируемость и корректность решений
